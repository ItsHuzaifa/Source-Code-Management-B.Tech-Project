User Manual

Available User Handles

  • 1- Admin(Master) Panel: Full System Control
  • 2- Faculty(Project Head) Panel: Raisoni Group of Institution Faculties
  • 3- Student(Member) Panel: Raisoni Group of Institution Students

User Handle description

Admin

  • Admin can able to handle all authorities
  • Admin can add, delete & update all the users in the system
  • Admin can manage repositories created by the students
  • Admin can all delete, update and add task or issues

Faculties

  • Faculties can view & assess code writtern by the students
  • Faculties can assign functional tasks to particular student & validate the same
  • Faculties can view the code commits made by the students in a timely manner
  • Faculties can track the progress of all the groups as a single view summary & can print the summary

Student

  • Student can checkout the repository of the assign project by the faculty
  • Student can download the repository and update the recent source code
  • Student can perform regular code commits as & when asked by the faculties
  • Student can write the cource code collaboratively & view the team work


  • We are the student of Raisoni Group of Institution Pune have designed online source code assessment cum management web application , which will help the students & faculties of RGI group for a better assessment of source code written by the students throughout their curricular of bachelors.
